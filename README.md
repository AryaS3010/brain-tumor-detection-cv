# brain-tumor-detection-cv
Brain Tumor Detection and MRI Analysis using Traditional Computer Vision Techniques

## Overview
A Computer Vision project for detecting brain tumors from MRI scans using traditional image processing and machine learning techniques.

## Features
- MRI Image Preprocessing
- Frequency Domain Enhancement
- Otsu Threshold Segmentation
- Morphological Operations
- Feature Extraction
- KNN Classification
- Tumor Severity Estimation
- Heatmap Visualization
- PDF Report Generation

## Technologies Used
- Python
- OpenCV
- Scikit-Learn
- NumPy
- Streamlit

Dataset used for this project:

- Achieved approximately 81–82% classification accuracy.
- Interactive MRI analysis using Streamlit.
